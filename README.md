# Snake Game
Project Title:
🐍 Snake Game using Python (Turtle Graphics)

Short Description:
This is a classic Snake Game developed using Python’s Turtle graphics library. The game features smooth controls, dynamic food generation, score tracking, and increasing difficulty as the snake grows. It also includes a high score system and continuous gameplay with screen wrap-around mechanics.

2️⃣ 🛠️ Technologies Used

    Python

    Turtle Graphics Library

Modules Used:

    turtle (for game UI & movement)

    random (for food positioning)

    time (for game speed control)

3️⃣ 📁 Data Source

    No external dataset used

    Fully logic-based game developed using Python

4️⃣ ⭐ Features / Highlights

    🎮 Arrow key controls for smooth movement

     🍎 Random food generation system

     📈 Score & High Score tracking

     ⚡ Increasing speed as score increases

     🔁 Screen wrap-around (snake appears from opposite side)

     🧱 Collision detection (self-collision resets game)

     🧩 Dynamic snake body growth
     
     ⏸️ Stop functionality using space key
     
👨‍💻 Custom credit display (“Created by Himanshu Ambulkar”)
